## Development

This section details the setup of a basic developmentment environment.

### Getting Started

**System Requirements**

- [Node.js][nodejs] v14

---

**Clone and Install Dependencies**

```console
> git clone https://github.com/dd060606/ReactMCLauncher.git
> cd ReactMCLauncher
> npm i
```

---

**Launch Application**

```console
> npm start
```

---

**Build Installers**

To build for your current platform.

```console
> npm run build
```
